# Project rules

## Research

Every research output (topic summaries, comparisons, technology surveys, Q&A dumps, reference material gathered while exploring a question) must be written to a file under `research/`. Do not leave research only in the chat transcript.

- One topic per file, kebab-case filename (e.g. `research/can-bus-extractable-data.md`).
- Include a short header with the question and date.
- Update the existing file instead of creating a new one if the topic already has a file.
